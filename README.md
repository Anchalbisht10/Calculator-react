## Calculator (React)

A responsive calculator application built using React, demonstrating component-based architecture and state management.

### Features
- Performs basic arithmetic operations
- Built using reusable React components
- Uses state to manage inputs and results
- Responsive and user-friendly interface

### Tech Stack
- React.js
- JavaScript
- CSS
